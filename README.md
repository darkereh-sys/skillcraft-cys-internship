# skillcraft-cys-internship

This repository consists of different types of cybersecurity related tasks performed during the Skillcraft Technology internship.

Details - 
Task 1: Caesar Cipher Tool : Developed an encryption and decryption program based on the classical Caesar cipher using Python.

Task 2: Image Encryption/Decryption Tool: Created a tool to encrypt and decrypt images, often involving pixel manipulation by swapping, XOR'ing or doing both using Python.

Task 3: Password Strength Checker/Assessor: Built an application (CLI or GUI) that evaluates the strength of a user's password based on length, use of uppercase/lowercase letters, numbers, and special characters.

Task 4: Keylogger Project: Developed a basic keylogger program designed to capture keystrokes and write them to a file for educational purposes (understanding monitoring mechanisms).
